# SC Racing - Trojan Network

A social networking platform for USC students with integrated gaming features.

## Features

- User authentication with USC email
- Social feed with posts, likes, and comments
- Profile management
- Integrated Cardinal Course game
- Leaderboard system

## Deployment

This project is deployed using GitHub Pages. The site is available at:
https://boxxelf.github.io/IYA-Hack26-SC-Racing/

## Setup

1. Clone the repository
2. Open `index.html` in a web browser

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for data persistence
